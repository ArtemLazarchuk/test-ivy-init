# TestIvyInit 

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy). 

Ivy is a web framework for building interactive web applications using C# and .NET.

## Run

```bash
ivy run
```

## Deploy

</p>

The button opens the Sliplane deploy flow. On GitHub, your repository is inferred from the page you open the link from. If you ran `ivy init` with a GitHub remote (or `IVY_REPO_URL`), the link may already include `repo` and `branch` query parameters so it works from anywhere.

```bash
ivy deploy
```
